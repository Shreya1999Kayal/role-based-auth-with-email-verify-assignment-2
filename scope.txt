Role-Based Authentication System with Email OTP Verification using Express.js and MongoDB


Project Objective:
Build a secure authentication system that allows users to register, verify their email using OTP, log in, and access protected routes based on assigned roles (e.g., admin, user).


 Deliverables:
·       Backend REST API with role-based access control

·       OTP-based email verification system

·       JWT-based authentication

·       MongoDB data models for user and OTP

·       Full API documentation (Postman collection or README)

Goals:

·       Create MongoDB connection

·       Build user registration and login endpoints

 Features:

Email OTP Verification System
 Goals:
